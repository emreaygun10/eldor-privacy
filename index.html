Gizlilik Politikası

Son Güncelleme: 07/11/2025

Eldor (“Uygulama”), Eldor şirket çalışanları ve stajyerlerinin şirket içi iş süreçlerine erişmesi için geliştirilmiş bir mobil uygulamadır. Bu gizlilik politikası, kişisel verilerin nasıl toplandığını, kullanıldığını, saklandığını ve paylaşıldığını açıklar.

Toplanan Veriler

  • Ad ve Soyad
  • E-posta adresi
  • Profil fotoğrafı
  • Telefon numarası
  • Personel ID / Kullanıcı ID
  • Giriş bilgileri (kullanıcı adı ve şifre)
  • Cihaz bilgileri (model, işletim sistemi vb.)
  • Uygulama hata logları (Firebase Crashlytics)

Verilerin Toplanma Amaçları

  • Kullanıcının kimliğini doğrulama
  • Bildirimlerin gönderilmesi
  • Uygulamanın güvenlik ve performansının artırılması
  • Hataların tespiti ve iyileştirme çalışmaları

Verilerin Saklanması

Veriler aşağıda belirtilen ortamlarda saklanır:

  • Firebase (cihaz verileri, crash logları)
  • Google Cloud Database (kişisel kullanıcı verileri)

Veriler yalnızca şirket içi yetkililerle paylaşılır. Üçüncü taraf paylaşımı yapılmaz (Firebase/Google dışındaki kurumlarla).

Veri Silme

Kullanıcılar, uygulama üzerinden hesaplarını silebilir. Silme işlemi sonrası kişisel veriler kalıcı olarak kaldırılır.

Çocuklara Yönelik Hizmet Değildir

Uygulama çocuklar için tasarlanmamıştır, yalnızca şirket çalışanları ve stajyerler tarafından kullanılabilir.

Privacy Policy (English)

Last Updated: 07/11/2025

Eldor (“the App”) is developed for internal use by Eldor employees and interns. This Privacy Policy explains how personal data is collected, used, stored, and shared within the app.

Data We Collect

  • Full Name
  • Email Address
  • Profile Photo
  • Phone Number
  • Employee ID / User ID
  • Login Credentials (username & password)
  • Device Information (model, OS, etc.)
  • Crash log data (Firebase Crashlytics)

How We Use the Data

  • User authentication
  • Sending notifications
  • Security and performance improvement
  • Detecting and resolving technical issues

Data Storage

Personal data is stored in the following environments:

  • Firebase (device data, crash logs)
  • Google Cloud Database (personal information)

Data is only shared internally. No third-party sharing occurs except with service providers (Firebase/Google).

Data Deletion

Users can delete their accounts within the app. All personal data will be permanently deleted afterward.

Not for Children

The app is not intended for children; it is only available to company employees and interns.
