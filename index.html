Gizlilik Politikası

Son Güncelleme: 07/11/2025

Eldor (“Uygulama”), Eldor şirket çalışanları ve stajyerlerinin şirket içi süreçlere erişebilmesi için geliştirilmiş bir mobil uygulamadır. Bu gizlilik politikası, kişisel verilerin nasıl toplandığını, kullanıldığını, saklandığını ve paylaşıldığını açıklar.

Toplanan Veriler

  • Ad ve Soyad
  • E-posta adresi
  • Profil fotoğrafı
  • Telefon numarası
  • Personel ID / Kullanıcı ID
  • Giriş bilgileri (kullanıcı adı ve şifre)
  • Cihaz bilgileri (model, işletim sistemi vb.)
  • Uygulama hata logları (Firebase Crashlytics)

Verilerin Toplanma Amaçları

  • Kullanıcının kimliğini doğrulama
  • Bildirimlerin gönderilmesi
  • Uygulamanın güvenlik ve performansının artırılması
  • Hataların tespiti ve iyileştirme çalışmaları

Verilerin Saklanması

Veriler aşağıdaki ortamlarda saklanır:

  • Firebase (cihaz verileri, crash logları)
  • Google Cloud Database (kişisel kullanıcı verileri)

Veriler yalnızca şirket içi yetkililerle paylaşılır. Firebase ve Google Cloud dışındaki üçüncü taraflarla paylaşım yapılmaz.

Üçüncü Taraf Hizmetleri

Uygulama, Firebase Crashlytics ve Firebase Cloud Messaging gibi hizmetleri kullanabilir. Bu servisler yalnızca performans analizi, bildirim gönderimi ve hata takibi amacıyla çalışır.

Veri Güvenliği

Veriler güvenli sunucularda, şifreleme ve erişim kontrol önlemleriyle korunur. Verilere yalnızca yetkili personel erişebilir.

Veri Saklama Süresi

Kişisel veriler yalnızca gerekli olduğu süre boyunca veya yasal yükümlülükler gereği saklanır. Süre dolduğunda veriler güvenli şekilde silinir.

Kullanıcı Hakları

Kullanıcılar, kişisel verilerine erişme, düzeltme veya silme hakkına sahiptir. Talepler smart.factory@niso.com.tr adresine e-posta gönderilerek iletilebilir.

Politika Değişiklikleri

Bu gizlilik politikası zaman zaman güncellenebilir. Önemli değişiklikler, uygulama içi bildirim veya e-posta yoluyla bildirilecektir.

Veri Silme

Kullanıcılar hesaplarını uygulama üzerinden silebilir. Silme işlemi sonrası tüm kişisel veriler kalıcı olarak kaldırılır.

Çocuklara Yönelik Hizmet Değildir

Uygulama çocuklar için tasarlanmamıştır; yalnızca Eldor çalışanları ve stajyerler tarafından kullanılabilir.

İletişim

Herhangi bir gizlilik talebi için:
E-posta: smart.factory@niso.com.tr

Privacy Policy

Last Updated: 07/11/2025

Eldor (“the App”) is a mobile application developed for internal use by Eldor employees and interns. This Privacy Policy explains how personal data is collected, used, stored, and shared.

Data We Collect

  • Full Name
  • Email Address
  • Profile Photo
  • Phone Number
  • Employee ID / User ID
  • Login Credentials (username & password)
  • Device Information (model, OS, etc.)
  • Crash log data (Firebase Crashlytics)

Purpose of Data Collection

  • User authentication
  • Sending notifications
  • Improving app security and performance
  • Identifying and fixing technical issues

Data Storage

  • Firebase (device data, crash logs)
  • Google Cloud Database (user data)

Data is shared only within the company. No third-party access occurs other than Firebase/Google services.

Third-Party Services

The App may use Firebase Crashlytics and Firebase Cloud Messaging for crash reporting and notifications.

Data Security

All personal data is protected using encryption, secure servers, and restricted access control.

Data Retention

Personal data is stored only as long as necessary or required by law. When no longer needed, data is securely deleted.

User Rights

Users have the right to access, correct, or delete their data. Requests can be sent to smart.factory@niso.com.tr.

Policy Updates

This policy may be updated periodically. Major updates will be communicated via in-app notice or email.

Account Deletion

Users can delete their accounts in the app. All personal data will be permanently removed after deletion.

Not for Children

This app is not intended for children. It is only accessible to Eldor employees and interns.

Contact

For any privacy inquiries:
Email: smart.factory@niso.com.tr

Política de Privacidade

Última Atualização: 07/11/2025

O Eldor (“Aplicativo”) foi desenvolvido para uso interno por funcionários e estagiários da empresa Eldor. Esta Política de Privacidade explica como os dados pessoais são coletados, utilizados, armazenados e compartilhados.

Dados Coletados

  • Nome completo
  • Endereço de e-mail
  • Foto de perfil
  • Número de telefone
  • ID de funcionário / ID de usuário
  • Credenciais de login (nome de usuário e senha)
  • Informações do dispositivo (modelo, sistema operacional, etc.)
  • Registros de falhas do aplicativo (Firebase Crashlytics)

Finalidade da Coleta de Dados

  • Autenticação do usuário
  • Envio de notificações
  • Melhoria da segurança e desempenho do aplicativo
  • Identificação e correção de falhas técnicas

Armazenamento de Dados

  • Firebase (dados do dispositivo, registros de falhas)
  • Banco de Dados do Google Cloud (dados pessoais)

Os dados são compartilhados apenas internamente. Nenhum acesso de terceiros é permitido, exceto serviços do Firebase/Google.

Serviços de Terceiros

O aplicativo pode usar Firebase Crashlytics e Firebase Cloud Messaging para relatórios de falhas e envio de notificações.

Segurança dos Dados

Os dados pessoais são protegidos por meio de criptografia, servidores seguros e controle restrito de acesso.

Retenção de Dados

Os dados pessoais são mantidos apenas pelo tempo necessário ou exigido por lei. Após esse período, são excluídos com segurança.

Direitos do Usuário

Os usuários têm o direito de acessar, corrigir ou excluir seus dados. Solicitações podem ser enviadas para smart.factory@niso.com.tr.

Alterações na Política

Esta política pode ser atualizada periodicamente. Alterações significativas serão comunicadas por meio de aviso no aplicativo ou e-mail.

Exclusão de Conta

Os usuários podem excluir suas contas diretamente no aplicativo. Após a exclusão, todos os dados pessoais serão removidos permanentemente.

Não Destinado a Crianças

O aplicativo não é destinado a crianças; é voltado apenas para funcionários e estagiários da Eldor.

Contato

Para solicitações relacionadas à privacidade:
E-mail: smart.factory@niso.com.tr
